# 📸 NeoImg ([https://img.bytexel.com](https://img.bytexel.com))

NeoImg 是一款基于浏览器的专业级图像处理工具，旨在提供极简的操作体验与强大的处理能力。所有核心图像操作（裁剪、缩放、压缩）均在**本地浏览器**完成，确保您的隐私数据永不上传服务器。

## ✨ 核心功能

- 🖼️ **智能裁剪 (Crop)**：支持自由拖拽选择区域，内置三分法构图辅助线。
- 📏 **精准缩放 (Resize)**：支持锁定宽高比，一键调整像素尺寸。
- 📉 **高效压缩 (Compress)**：毫秒级本地压缩，支持实时预览预计文件大小。
- 🔄 **格式转换 (Convert)**：支持在 PNG, JPEG, WebP 之间无缝切换。
- 🤖 **AI 智能增强**：集成 Google Gemini API，自动生成：
  - 符合 SEO 标准的描述性 **Alt Text**。
  - 智能内容 **标签 (Tags)**。
  - 语义化的 **建议文件名**。
- 🔒 **隐私至上**：采用客户端处理技术，图片不会被上传到任何后端存储。

---

## 🚀 技术栈

- **Frontend**: React 19 (Hooks, Context)
- **Styling**: Tailwind CSS (Glassmorphism UI)
- **Build Tool**: Vite
- **AI Engine**: Google Generative AI (Gemini 3 Flash)
- **Image Processing**: HTML5 Canvas API
- **Deployment**: GitHub Pages + Custom Domain
